# Student Management System

A complete Student Management System built with Java Spring Boot, MySQL, Thymeleaf, HTML, CSS, and JavaScript.

## Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- HTML5
- CSS3
- JavaScript
- MySQL

## Features
- Add student
- View student list
- Search student by name or course
- Update student details
- Delete student details
- Server-side validation
- Responsive user interface

## Run Project
1. Create database using `schema.sql` or let Spring Boot create it.
2. Update MySQL username and password in `src/main/resources/application.properties`.
3. Run MySQL server.
4. Start with `mvn spring-boot:run`.
5. Open `http://localhost:8080/students`.
